# BP BA Analysis Agent

This directory contains the BP BA / Sales BP analysis agent proof-of-concept.

## Contents

- `src/bp_ba_agent/`: core Python agent, analysis workflow, semantic layer, real-data demo logic, traffic review logic, and workflow web prototype backend.
- `v2_workbench/`: newer multi-agent workbench prototype with backend and frontend.
- `skills/bp-ba-sales-analysis/`: Codex skill definition for sales analysis topics and field mappings.
- `docs/`: product, roadmap, and slide content documentation.
- `examples/`: demo entry scripts.
- `tests/`: pytest coverage for the main agent capabilities.
